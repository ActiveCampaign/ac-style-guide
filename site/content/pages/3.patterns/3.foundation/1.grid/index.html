Principles

Cohesive

Elements should flow naturally within the grid’s defined boundaries. Padding and spacing should be consistent and uniform.

 

Responsive

Screens should adapt to a user’s viewing device to maximize content visibility and increase usability.

 

Predictive

Consistent element sizing and placement creates a unified experience allowing the user to feel familiar with each interface.
